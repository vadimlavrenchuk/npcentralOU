# Быстрый старт - JWT авторизация

## ⚡ Первый запуск

### 1. Создать базу и дефолтного админа
```bash
cd backend
npm run seed
```

### 2. Запустить backend
```bash
cd backend
npm run dev
```

### 3. Запустить frontend
```bash
npm run dev
```

## 🔑 Вход в систему

**Первый администратор (только локальная разработка):** создаётся скриптом `npm run seed`; логин и пароль выводятся в терминал. Сразу смените пароль и не публикуйте учётные данные.

## 👥 Создание пользователей

1. Войдите как `admin`
2. Откройте **"Управление пользователями"** в меню
3. Нажмите **"+ Создать пользователя"**
4. Заполните форму:
   - Логин (уникальный)
   - Пароль (минимум 6 символов)
   - Полное имя
   - Роль
5. Нажмите **"Создать"**

## 🔐 Роли и права

| Роль | Права |
|------|-------|
| **Admin** | Полный доступ ко всему + управление пользователями |
| **Chief Mechanic** | Управление заказами, складом, оборудованием, сотрудниками |
| **Accountant** | Просмотр склада, финансовые отчёты |
| **Mechanic** | Создание заказов, просмотр склада и оборудования |

## 🛡️ Безопасность

- ✅ Новые пользователи создаются только админом
- ✅ Нет самостоятельной регистрации
- ✅ Пароли хешируются bcrypt
- ✅ JWT токены с ограниченным сроком действия
- ✅ Возможность блокировки аккаунтов

## 📋 Управление пользователями (только Admin)

### Изменить роль
Выберите новую роль из выпадающего списка - изменения применятся автоматически

### Заблокировать/Разблокировать
Нажмите иконку 🔒 или 🔓

### Удалить пользователя
Нажмите иконку 🗑️ и подтвердите удаление

## ⚠️ Важно

- Нельзя заблокировать или удалить свой собственный аккаунт
- Смените пароль admin после первого входа
- Используйте сильные пароли для всех пользователей

## 🚀 Готово!

Система готова к использованию. Все операции с пользователями доступны через веб-интерфейс.
